# ☕ Cupping Pro

**Профессиональный бланк для каппинга кофе по протоколу Specialty Coffee Association (SCA).**  
Полностью клиентское веб-приложение, работающее без сервера и установки.  
Создано для бариста, обжарщиков, Q-грейдеров и всех, кто хочет оценивать кофе по международным стандартам.

---

## ✨ Возможности

- 📋 **Создание и ведение форм каппинга** с полями согласно протоколу SCA
- 🌗 **Тёмная / светлая тема** – выбор сохраняется в браузере
- 📊 **Автоматический подсчёт баллов** с учётом дефектов (Taint/Fault)
- 💾 **Сохранение в локальное хранилище** – данные не теряются при закрытии браузера
- 📄 **Экспорт в PDF и JSON**
  - PDF с полным форматированием и итоговой классификацией
  - JSON для обмена формами между устройствами
- 🤝 **Импорт форм из JSON-файлов**
- 📤 **Поделиться PDF** – через Web Share API или скачивание файла
- 🧪 **Дополнительные технические поля**
  - Дата обжарки кофе
  - Минерализация воды (TDS)
  - Температура заваривания
- 📱 **Адаптивный дизайн** – удобно на смартфонах, планшетах и ПК
- ♿ **Доступность** – плавная анимация, поддержка reduced motion, прокрутка и уведомления

## 📐 Оценка по шкале SCA

Приложение реализует полную шкалу оценки арабики:

- **Аромат / Букет / Послевкусие / Кислотность / Тело / Баланс** – шаг 0.25 балла (6.00 – 10.00)
- **Сладость / Однородность / Чистая чашка** – по 2 балла за каждую из 5 чашек (максимум 10)
- **Дефекты** – лёгкие (Taint, -2) и серьёзные (Fault, -4) с учётом количества чашек
- **Итоговая классификация** – от «Выдающийся» (90+) до «Ниже Speciality» (<80)

Подробное описание протокола и последовательность действий встроены в главный экран приложения.

## 🚀 Как использовать

1. Откройте файл `Cupping Pro (Beta) — копия.html` в любом современном браузере (Chrome, Firefox, Safari, Edge).
2. На главном экране выберите:
   - **Новая форма** – создаёт чистый бланк
   - **Сохранённые** – просмотр и управление ранее заполненными формами
   - **Импорт** – загрузите ранее экспортированный JSON-файл
3. Заполните поля информации, введите оценки, отметьте дефекты.
4. Для каждой оценки (кроме трёх бинарных параметров) используйте кнопки с шагом 0.25, для них – кнопки‑чашки (5 штук).
5. При необходимости добавьте заметки к каждому параметру.
6. Итоговый балл и классификация обновляются автоматически.
7. Сохраните форму нажав «Сохранить» (или при выходе с несохранёнными данными появится запрос).
8. Экспортируйте в PDF / JSON или поделитесь PDF напрямую (на устройствах с поддержкой Web Share).
